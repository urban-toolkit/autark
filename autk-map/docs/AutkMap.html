Class

AutkMap

AutkMap(canvas, autoResizeopt)

The main autark map class. `AutkMap` encapsulates the core logic for initializing and rendering a map on a given HTML canvas element. It manages the camera, map rendering, map layers, and user interactions through keyboard, mouse, and map events.

autoResize boolean <optional>

Example
const canvas = document.getElementById('map-canvas') as HTMLCanvasElement;
const map = new AutkMap(canvas);
await map.init(boundingBox);
map.loadGeoJsonLayer('roads', LayerType.OSM_ROADS, geojsonData);

# loadGeoJsonLayer(layerName, typeLayer, geojson)

Loads a GeoJSON layer into the map. This method creates a layer based on the provided GeoJSON data and adds it to the map's layer manager. Supported OSM layer types include: - OSM_COASTLINE - OSM_WATER - OSM_PARKS - OSM_ROADS - OSM_BUILDINGS Custom layers can also be loaded with types: - CUSTOM_FEATURES_LAYER - CUSTOM_LINES_LAYER - CUSTOM_GRID_LAYER
Parameters:
Name Type Description
